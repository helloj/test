T00 — 純直線播放 預期：按順序播放 8 個音，共 2 小節，結束。

T01 — D.C. 最簡形式 預期播放順序： C D E F → G A B c → z（D.C.） 跳回開頭：C D E F → G A B c → 結束（第二次不重複跳轉）

T02 — D.C. al Fine 預期播放順序： C D E F → G A B c（!fine! 在此小節開頭）→ z（D.C.） 跳回頭：C D E F → 到 !fine! 位置停止

T03 — D.S. 最簡形式 預期播放順序： C D E F → G A B c（!segno!）→ E F G A → z（D.S.） 跳到 !segno!：G A B c → E F G A → 結束（第二次不再跳）

T04 — D.S. al Fine 預期播放順序： C D E F → G A B c（!segno! 在頭，!fine! 在 B 前）→ E F G A → z（D.S.） 跳到 !segno!：G A → 到 !fine! 位置停止

T05 — D.C. al Coda（最常見形式） 預期播放順序： C D E F → G A B c → z（D.C. al Coda，跳回頭） C D E F → G A B c → 遇到 !coda! 分歧點跳出 !coda!：e f g a → 結束

T06 — D.S. al Coda（標準歌曲結構） 預期播放順序： C D E F → G A B c（!segno!）→ E F G A → z（D.S. al Coda） 跳到 !segno!：G A B c → E F G A → coda 分歧點跳出 !coda!：e f g a → 結束

T07 — 一段反覆後接 D.S. al Coda 預期播放順序： C D E F → G A B c（房 1） C D E F → G A B G（房 2） e d c B（!segno!）→ A B c d → z（D.S. al Coda） 跳到 !segno!：e d c B → A B c d → 分歧點跳出 !coda!：G A B c → 結束

T08 — !segno! 在第一小節開頭（等同 D.C.） 預期：與 T01 D.C. 行為相同——跳回曲首（因為 !segno! 就在第一個音符上）。

T09 — 多聲部 D.S. al Coda 預期：兩個聲部同步跳轉，coda 段兩聲部均正確播出。

T10 — !coda! 緊接在 !D.S.alcoda! 後（零距離 coda） 預期： C D E F → G A B c → 遇到 D.S. al Coda 跳到 !segno!：C D E F → G A B c → 到達 coda 分歧點 進入 coda：e f g a → 結束

T11 — 分歧點 coda 與降落點 coda 分開標記 預期播放順序： C D E F → G A（!segno!）→ B c（第一個 !coda! 在 B 前） E F G A → z（D.S. al Coda） 跳到 !segno!：G A → 遇到第一個 !coda!（分歧點）跳出 進入第二個 !coda!（降落點）：e f g a → 結束

T12 — 兩個O＋反覆＋fine 結構：!segno! 在中段，第一個 O 在 [1 之前當分歧點， 第二個 O 與 !D.S.! 在同一音符（降落點），最後有 !fine!

T13 — multiple repeat（[1-3 / [4,5 / ["last time"]） 預期播放順序： [段落 A] C D E F | G F E D 反覆 2 次（房1：C2 G2；房2：C2 C2） [段落 B] c3 z（直線過門） [段落 C] C D E F 反覆 6 次： 第 1–3 次接房 1-3（c d e f） 第 4–5 次接房 4,5（C2 G2） 第 6 次接房 "last time"（C G C z）→ 結束

T14 — triple repeat（各種非標準小節線綜合壓力測試） 預期：不崩潰、不無限循環；各段反覆次數依小節線冒號數決定 （:: 重複 2 次，:::| 重複 3 次）。

T15 — D.C. 在 repeat 括弧內（白牡丹同結構） 預期播放順序： A（前奏）→ B !D.C.!（第一次，pass，不跳）→ C → repeat 回 B B !D.C.!（第二次，enable，跳曲首）→ A → B → C → 結束

T16 — D.C. 在 repeat 括弧內，無前奏段（|: 就是曲首） 預期播放順序： |: A !D.C.!（第一次，pass）→ B → repeat 回 A A !D.C.!（第二次，enable，跳曲首）→ A → B → 結束

T17 — D.S. 在 repeat 括弧內 預期播放順序： A → !segno! B → C !D.S.!（第一次，pass）→ D → repeat 回 B B → C !D.S.!（第二次，enable，跳 segno）→ B → C → D → 結束

T18 — D.C. al Fine 在 repeat 括弧內 預期播放順序： !fine! A → B !D.C.alfine!（第一次，pass）→ C → repeat 回 B B !D.C.alfine!（第二次，enable，跳曲首）→ 到 !fine! 停止

T19 — D.C. al Coda 在 repeat 括弧內 預期播放順序： A → !coda! B → C !D.C.alcoda!（第一次，pass）→ D → repeat 回 C C !D.C.alcoda!（第二次，enable，跳曲首）→ A → B 遇 coda 分歧點跳出 !coda!（降落）：e f g a → 結束

T20 — D.C. 在 repeat 內，且 repeat 有 volta bracket 結構：|: A |1 B !D.C.! :|2 C | 預期（慣例 A，DC 後 repeat 從頭計數）： 前奏 → [1房 pass] → [1房 DC跳曲首，repn/repv 重置] → 前奏 → [1房] → [2房] → 結束 驗證：_isInsideRepeat=true，第一次 pass；跳回後 po.repn=false / po.repv=1。

T21 — D.C. 在 repeat 外（repeat 之後），對照組 結構：|: A :| B !D.C.! | 預期播放順序： |: A :| repeat → A → B !D.C.!（立即跳，不需 pass）→ 曲首 A → repeat → A → B → 結束 驗證：D.C. 在 :| 之後，_isInsideRepeat 應為 false，第一次即跳（T01 同類）。
